# DIO Music

## Descrição do Projeto
Aplicação para gerenciamento de músicas em tempo de execução via console.

### Testar projeto

```
# Clonar repositorio
$ git clone https://gitlab.com/mariz5g/dio_music.git

# Acessar pasta
$ cd dio_music

# Rodar
$ dotnet run
```
